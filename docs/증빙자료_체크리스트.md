## 2. 테스트 입력 재현 스크린샷 (평가문항 1 대응)

정상/빈값/긴 입력 2~3개를 실제로 넣어보고 캡처하세요.
- [ ] 정상 입력 → 정상 결과 표시
  ![정상 입력 결과](screenshots/test-cases/normal-input.png)
- [ ] 빈 입력 또는 실패 유도 상황 → 안내 문구가 뜨는 화면
  ![빈 입력/실패 안내](screenshots/test-cases/empty-input.png)
- [ ] (선택) 200자 이상의 긴 입력 → 정상 처리 또는 지연 안내 문구
  ![긴 입력 처리](screenshots/test-cases/long-input.png)

> ⚠️ 지난 대화에서 짚었던 것처럼, 현재 실패 시 안내 문구가 안 뜨는 상태라면 이 스크린샷을 찍을 때 그 사실을 미리 확인해두는 게 좋아요. 필요하면 이후 파일 수정 단계에서 같이 고칠게요.
